# Deskripsi Sistem

## Nama Sistem

Sistem Informasi Bengkel Motor

## Tujuan

Sistem ini dibuat untuk membantu pengelolaan data pelanggan, servis kendaraan, stok suku cadang, dan transaksi pembayaran sehingga pelayanan bengkel menjadi lebih cepat dan efisien.

## Pengguna

- Admin
- Mekanik
- Kasir
- Pelanggan

## Fitur Utama

- Login
- Data Pelanggan
- Data Kendaraan
- Booking Servis
- Data Servis
- Stok Suku Cadang
- Transaksi Pembayaran
- Laporan Servis
