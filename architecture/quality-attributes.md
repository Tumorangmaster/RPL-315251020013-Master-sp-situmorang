# Quality Attributes

## Performance

Sistem mampu memproses data servis dan transaksi dengan cepat.

## Security

Data pelanggan dan transaksi dilindungi menggunakan autentikasi pengguna dan enkripsi password.

## Usability

Antarmuka dibuat sederhana sehingga mudah digunakan oleh admin, mekanik, dan kasir.

## Reliability

Sistem mampu beroperasi dengan stabil tanpa sering mengalami gangguan.

## Scalability

Sistem dapat dikembangkan untuk mendukung banyak cabang bengkel.
