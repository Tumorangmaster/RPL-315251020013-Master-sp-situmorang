# Arsitektur Sistem

## Pola Arsitektur

Layered Architecture

## Lapisan Sistem

### Presentation Layer

Menampilkan antarmuka aplikasi kepada pengguna.

### Business Layer

Mengelola proses login, booking servis, pencatatan servis, pengelolaan stok suku cadang, dan transaksi pembayaran.

### Data Access Layer

Menghubungkan aplikasi dengan database.

### Database Layer

Menyimpan seluruh data pelanggan, kendaraan, mekanik, suku cadang, dan transaksi.

## Alasan Pemilihan

Layered Architecture dipilih karena mudah dikembangkan, mudah dipelihara, serta memisahkan setiap fungsi sistem menjadi beberapa lapisan.
